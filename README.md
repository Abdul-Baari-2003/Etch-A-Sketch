# Etch-A-Sketch 🎨

A fun and interactive **Etch-A-Sketch** game built using **HTML, CSS, and JavaScript**. Users can draw on a grid-based board with different color modes and customizable sizes.

## 📌 Features
- 🎨 **Customizable Grid Size**: Users can set the board size (max 100x100).
- 🌈 **Color Modes**:
  - Black color drawing
  - Random color mode for vibrant effects
- 🖱️ **Click-to-Draw Toggle**: Click anywhere to enable/disable drawing.
- 🔄 **Reset Button**: Clears the board instantly.

## 🛠️ Technologies Used
- **HTML** - Structure of the webpage.
- **CSS** - Styling and grid layout.
- **JavaScript** - Functionality and user interactions.
- **Bootstrap** - Styling for buttons and layout enhancements.

## 🚀 How to Use
1. **Clone the repository** or download the project files.
2. Open `index.html` in a web browser.
3. Click **Select** to set the grid size.
4. **Click anywhere** to start/stop drawing.
5. Choose between **Black** or **Random** color mode.
6. Click **Reset** to clear the board.
